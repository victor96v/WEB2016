Features

  • Navigation options: arrows & bullet anchors
  • Full responsive.
    • Simple jQuery Slider assumes no styling, giving you complete control.
  • Autoplay
  • Can listen to hashchange and link to specific slides
  • Slides may contain ANY content.
  • Callback API
  • Supported across all major browsers.

Initiate With Javascript

Below is an example of how to initialize a slideshow with autoplay, bullets, arrows & a callback to add a FontAwesome class to the arrows.

window.slider = $('#slide');
window.slider.simpleSlide({
    'auto':true,
    'bullets':true,
    'slideSelector':'img',
    'onInitiate':function(){
        window.slider.find('a.arrow.prev').addClass('fa fa-hand-o-left');
        window.slider.find('a.arrow.next').addClass('fa fa-hand-o-right');
    }
});

 

$.simpleSlide(options)

Description: Creates a custom slideshow from the specified element.

 

options

A map of options to pass to the $.simpleSlide() method.

  • auto (default false)
    • Type: Boolean
    • Determines whether to automate the slideshow
  • autoTime (default 400)
    • Type: Number
    • The time between slides in an automated slideshow
  • autoInteraction (default false)
    • Type: Boolean
    • Determines whether or not to continue automatically playing slideshow after a user interacts (with bullets or arrows).
  • arrows (default true)
    • Type: Boolean
    • Determines whether or not to append arrow <a /> elements to the slideshow
  • bullets (default false)
    • Type: Boolean
    • Determines whether or not to append bullets to the slideshow
  • transitionTime (default 500)
    • Type: Number
    • The amount of time it takes to transition between slides
  • hashChange (default true)
    • Type: Boolean
    • Determines if the slideshow should listen to 'hashchange' event. Hash format example: #slide-0
  • slideSelector (default '.slide')
    • Type: String
    • Selector string for slide elements

 

event callbacks

Functions that can be fired when different events in the slideshow occur

  • onInitiate()
    • Type: Function()
    • Occurs when the slideshow is initiated
  • onBeforeSlide()
    • Type: Function()
    • Occurs before the slide changes
  • onSlideStart()
    • Type: Function()
    • Occurs when a new slide appears
  • onAfterSlide()
    • Type: Function()
    • Occurs after a new slide finishes animating
  • onBulletClick()
    • Type: Function()
    • Occurs when a bullet is clicked
  • onArrowClick()
    • Type: Function()
    • Occurs when either arrow element is clicked
  • onPreviousClick()
    • Type: Function()
    • Occurs when the 'previous' arrow is clicked
  • onNextClick()
    • Type: Function()
    • Occurs when the 'next' arrow is clicked
